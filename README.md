# PCCCS495 – Term II Project

## Project Title

--- Hotel Booking System

## Problem Statement (max 150 words)

--- Hotels often manage room reservations manually or through poorly organized systems, leading to booking conflicts, inefficient room allocation, and difficulty maintaining customer records. Customers require a fast and reliable system to check room availability, make bookings, and cancel reservations without errors.

Most basic systems lack structured design and fail to properly model real-world entities such as customers, rooms, and bookings. This project aims to solve these issues by developing a Hotel Booking System using Object-Oriented Programming principles in Java. The system models hotel operations through interacting classes and objects, ensuring modularity, scalability, and maintainability.

The application will allow users to manage rooms, customers, and reservations efficiently while demonstrating core OOP concepts such as abstraction, inheritance, encapsulation, and polymorphism.

## Target User

---Hotel reception staff
Small hotel owners
Students learning booking management systems

## Core Features

- Add and manage customer details
- Book rooms based on availability
- Automatic room status update (Available/Booked)
